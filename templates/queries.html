Few Queries

Selection Query using order by

Order by isused to sort the table based on specific column in ascending or descending order. select id,player_api_id,Player_name,height from player order by player_api_id desc;

Selection Query using join

Joining two table using join Select distinct(player.id),player.player_name,player_attributes.player_api_id,player.player_fifa_api_id, player_attributes.defensive_work_rate from player join player_attributes on player.player_fifa_api_id = player_attributes.player_fifa_api_id;

Selection Query using group by

groups values in the table based upon specific column select buildupplayspeedclass,count(buildupplayspeedclass) as team_number from team_attributes group by buildupplayspeedclass;

Selection Query using group by,having and order by

select buildupplayspeed,count(buildupplayspeed) as team_number from team_attributes group by buildupplayspeed having avg(team_attributes.buildupplayspeed)>55 order by buildupplayspeed desc;

Insertion Query

Insertion of specific values into rows in the order of their column data types Use a select query to check if the details are inserted

Deletion Query

Delete Query is used to delete specific rows in the table use select statement to check where deletion has been performed

Updation Query

Update query to update values of any specific columns of a table use select statement to check where update has been performed

Alteration Query

Alter query can drop columns or modify column types and also add constraints. use select statement to check where alter has been performed to add or drop columns.
